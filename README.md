# Jikesh Portfolio

This is a professional portfolio website for **Jikesh**, built with Next.js, React, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Development
First, run the development server:

```bash
npm run dev
```

Open [http://localhost:301](http://localhost:301) with your browser to see the result.

## Documentation
For a detailed overview of the project architecture, tech stack, and features, please refer to the [DOCUMENTATION.md](./DOCUMENTATION.md) file.

## Key Features
- **Modern Responsive Design**: Built with Tailwind CSS 4.
- **Dynamic Content**: Projects and products are managed via JSON data files.
- **Admin Dashboard**: Secure access to manage products and view messages.
- **Netlify Integration**: Optimized for deployment on Netlify with Blobs support for persistent storage.
